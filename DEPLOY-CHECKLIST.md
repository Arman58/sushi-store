# Деплой на прод — чеклист (изменения от 2026-07-03)

## Что уходит в этот релиз
1. **Редизайн витрины** — шапка (поиск-пилюля, локация, корзина, сердечко), hero, фото-карточки категорий, карточки товаров (тёмный «Хит», контурный «+», цена → старая цена).
2. **Избранное** — store (`ew_favorites`, совместим со старым форматом), страница `/favorites`, `GET /api/products/by-ids`, бейджи в шапке и bottom-nav.
3. **Отзывы** — модели `Review`/`ReviewVote`, рейтинг-агрегаты на товаре, API CRUD + «Полезно», UI на странице товара, JSON-LD aggregateRating.
4. **Админка** — страница «Категории» (фото: загрузка/замена/удаление), таблица товаров (поиск/фильтры/сортировка/пагинация).

## Порядок деплоя

### 1. Локально перед пушем
```bash
npm run prisma:migrate        # применит 20260703120000_reviews_and_category_image к БД из .env.local
npx prisma generate           # перегенерирует клиент (уберёт текущие 38 ошибок tsc)
npx tsc --noEmit              # должно быть 0 ошибок
npx eslint src                # чисто (проверено по изменённым файлам)
npm run dev                   # smoke-тест, см. п.3
```

⚠️ `prisma migrate dev` спросит имя — миграция уже написана руками в
`prisma/migrations/20260703120000_reviews_and_category_image/migration.sql`,
поэтому корректнее: `dotenv -e .env.local -- npx prisma migrate deploy` (без создания новой).

### 2. Прод
- Build-команда уже содержит `prisma migrate deploy` → миграция применится сама при деплое на Vercel.
- Новых env-переменных **нет** (Cloudinary и DATABASE_URL уже настроены).

### 3. Smoke-тест после деплоя
- [ ] Главная: категории с фото, «Смотреть все», карточки без рейтинга (отзывов ещё нет — блок скрыт).
- [ ] Мобильный вид (≤600px): строка поиска под шапкой, локация видна, bottom-nav 5 пунктов.
- [ ] Сердечко на карточке → счётчики в шапке/bottom-nav → `/favorites` (пустое состояние и с товарами).
- [ ] Страница товара: секция «Отзывы», создать/изменить/удалить отзыв (нужен логин), «Полезно» с чужого аккаунта.
- [ ] Бейдж «Проверенная покупка» — у пользователя с заказом в статусе DONE с этим блюдом.
- [ ] Рейтинг появился на карточках (после первого отзыва) и в `<script type="application/ld+json">`.
- [ ] Админка → Категории: загрузить/заменить/убрать фото (≤5 МБ), витрина обновилась (ISR 60 сек).
